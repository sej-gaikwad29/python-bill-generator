# Python Bill Generator

A desktop application built using Python that allows users to generate
professional PDF invoices.

## Features
- GUI built using Tkinter
- Dynamic invoice item entry
- Scrollable table for multiple rows
- Automated PDF invoice generation
- Rupee symbol support
- Multi-page invoice handling
- Standalone executable using PyInstaller

## Technologies Used
- Python
- Tkinter
- ReportLab
- PyInstaller
